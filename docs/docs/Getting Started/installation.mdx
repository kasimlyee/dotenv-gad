# Installation

This guide explains how to install **dotenv-gad** and integrate it into your Node.js or TypeScript project.



## Requirements

Before installing, ensure your environment meets the following requirements:

* **Node.js** â‰¥ 18
* **npm**, **yarn**, or **pnpm**
* A JavaScript or TypeScript project

> `dotenv-gad` works with CommonJS and ES Modules.



Install `dotenv-gad` using your preferred package manager:


<Tabs groupId="package-manager">
  <TabItem value="npm" label="npm" default>

    ```bash
    npm install dotenv-gad
    ```

  
</TabItem>

<TabItem value="yarn" label="yarn">

    ```bash
    yarn add dotenv-gad
    ```

  
</TabItem>

<TabItem value="pnpm" label="pnpm">

    ```bash
    pnpm add dotenv-gad
    ```

  
</TabItem>
</Tabs>

## Verify Installation

After installation, verify that the package is available:

```bash
npx dotenv-gad --help
```

You should see the list of available CLI commands.
